# 00185fm

It's a Web Radio App that allows you to curate your own radio station with a focus on archiving materials.

## Features

- Admin Panel 👨‍💻
- Live Preview 💻
- Search 🔍
- Pagination 🔢
- Infinite Scroll 📱
- Playlists 🎶
- Tags 🏷️
- Rich Text Editor 🎨
- Access Control 🔒

## Technologies

- **Framework**: Next.js 15
- **CMS**: Payload 3.0
- **Database**: MongoDB
- **Storage Adapter**: Minio S3
